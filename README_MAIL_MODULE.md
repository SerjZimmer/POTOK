# Модуль "Почта" для POTOK - Итоговый README

## 🎯 Что создано

Модуль "Почта" для проекта POTOK успешно создан и интегрирован в основное приложение. Реализована полная архитектура backend и базовый UI frontend.

## 🏗️ Архитектура

### Backend (Go)
```
backend/internal/mail/
├── model/
│   └── types.go          # Все модели данных
├── provider/
│   ├── provider.go       # Интерфейсы провайдеров
│   ├── gmail.go         # Gmail провайдер (заглушка)
│   ├── imap.go          # IMAP провайдер (заглушка)
│   └── factory.go       # Фабрика провайдеров
├── service/
│   ├── accounts.go      # Сервис управления аккаунтами
│   ├── threads.go       # Сервис работы с тредами
│   └── send.go          # Сервис отправки сообщений
└── http/
    └── router.go        # HTTP роутер с Chi
```

### Frontend (Flutter)
```
frontend/lib/features/mail/
├── domain/
│   └── entities.dart    # Доменные сущности
└── presentation/
    ├── mail_screen.dart # Основной экран с вкладками
    ├── inbox_view.dart  # Экран входящих
    └── accounts_view.dart # Экран аккаунтов
```

## 🚀 Запуск

### 1. Backend (основной сервер)
```bash
cd backend
make run-backend
```

Сервер запустится на порту 8080 и будет доступен модуль "Почта" по адресу:
- `http://localhost:8080/v1/mail/*`

### 2. Отдельный сервер модуля "Почта"
```bash
cd backend/cmd/mail-server
go run main.go
```

Сервер запустится на порту 8081.

### 3. Frontend
```bash
cd frontend
flutter run -d chrome  # Web
# или
flutter run -d macos   # macOS
```

## 📱 Использование

### В приложении
1. Запустите Flutter приложение
2. В нижней навигации появится вкладка "Почта"
3. Нажмите на вкладку "Почта"
4. Увидите экран с вкладками: Входящие, Отправленные, Черновики, Архив, Спам, Корзина

### Добавление аккаунта
1. На экране "Почта" нажмите кнопку "+" в AppBar
2. Выберите провайдера (Gmail, Outlook, Яндекс, Mail.ru, IMAP)
3. Введите email и отображаемое имя
4. Нажмите "Добавить"

## 🔧 API Endpoints

### Аккаунты
- `GET /v1/mail/accounts` - список аккаунтов
- `POST /v1/mail/accounts` - создание аккаунта
- `GET /v1/mail/accounts/{uid}` - получение аккаунта
- `PATCH /v1/mail/accounts/{uid}` - обновление аккаунта
- `DELETE /v1/mail/accounts/{uid}` - удаление аккаунта

### Треды
- `GET /v1/mail/accounts/{uid}/threads` - список тредов
- `GET /v1/mail/threads/{id}` - получение треда

### Отправка
- `POST /v1/mail/send` - отправка сообщения
- `POST /v1/mail/drafts` - создание черновика

### OAuth
- `POST /v1/mail/oauth/{provider}/start` - начало OAuth
- `POST /v1/mail/oauth/{provider}/callback` - завершение OAuth

## 🐛 Устранение неполадок

### Частые проблемы
1. **Сервер не запускается**: проверьте порты 8080/8081
2. **Ошибки компиляции**: убедитесь, что Go 1.22+ и Flutter 3.x
3. **UI не отображается**: проверьте импорты в `app/main.dart`
